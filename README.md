## Run Terminal(Ubuntu)

Update dependencies

```bash
  apt update
```

Clone the project

```bash
  git clone https://github.com/uwtechshow-official/Queen-Spriky-WhatsApp-Bot
```

Go to the project directory

```bash
  cd Queen-Spriky-WhatsApp-Bot
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
